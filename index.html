How to add photos

Simple setup

  1. Place all birthday photos inside /images.
  2. Place your background song at /music/birthday-song.mp3.
  3. Run powershell -ExecutionPolicy Bypass -File .\scripts\update-image-manifest.ps1.
  4. Reload the page. The slideshow and gallery will use every image found in the folder.
